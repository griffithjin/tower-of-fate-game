## 시작하기

### 1. 계정 등록
- 타워 오브 페이트 공식 사이트 방문
- "가입" 버튼 클릭
- 이메일과 비밀번호 입력
- 이메일 주소 인증
- 프로필 완성

### 2. 테마 선택
- 설정 → 테마 이동
- 사용 가능한 테마 중 선택:
  - 클래식 타워
  - 모던 스카이스크래퍼
  - 미스티컬 캐슬
  - 스페이스 스테이션
  - 네이처 하모니

### 3. 카드 덱 선택
- 카드 컬렉션 이동
- 4가지 다른 덱 중 선택
- 각 덱에는 고유한 능력과 전략이 있음
- 획득한 카드로 덱 커스터마이즈

### 4. 게임 참여
- "지금 플레이"로 빠른 매치
- 또는 "토너먼트 참가"로 경쟁 플레이
- 매칭 완료 대기
- 모든 플레이어 준비 완료 시 게임 시작

## 게임 규칙

### 기본 규칙
- **4세트 카드** (총 208장)
- **13명의 가디언**, 각각 13장의 카드
- 가디언 카드를 매치하여 타워를 오르기
- 분노 카드 (패널티 카드) 피하기
- A층에 먼저 도달한 플레이어 승리

### 타워 구조
```
A층  - 🏆 승리 층
K층
Q층
J층
10층
9층
8층
7층
6층
5층
4층
3층
2층
1층  - 시작점
```

### 카드 매칭
- 현재 가디언과 매치되는 카드 내기
- 성공 매치 시 1층 상승
- 잘못된 카드는 패널티 유발 가능
- 특수 카드는 고유한 효과 보유

### 분노 카드
- 빨간색 표시 카드는 "분노 카드"
- 분노 카드를 뽑으면 패널티:
  - 1턴 상실
  - 1층 하강
  - 카드 2장 버리기
- 분노 카드를 피하는 전략 사용!

## 게임 모드

### 🎮 솔로 챌린지
- AI 상대와 대전
- 전략 연습
- 챌린지 완료 시 보상 획득
- 3개의 난이도: 쉬움, 보통, 어려움

### 👥 팀 배틀
- 2-4인 팀 구성
- 협력 게임플레이
- 자원과 전략 공유
- 팀으로 승리

### 🏆 토너먼트
- 정기 개최 경쟁 이벤트
- 참가비와 상금 풀
- 랭킹 시스템
- 주간 및 월간 챔피언십

### 🔥 연승 모드
- 연속 승리
- 연승 복수 보상
- 1회 패배 = 연승 리셋
- 리더보드 랭킹

### 🎲 퀵 매치
- 즉시 매칭
- 랜덤 상대
- 5분 게임
- 짧은 휴식에 적합

## 팁과 전략

### 초보자 팁
1. **카드 타입 배우기**
   - 일반 카드 (1-10)
   - 페이스 카드 (J, Q, K, A)
   - 특수 액션 카드
   - 분노 카드 (피하기!)

2. **상대 관찰**
   - 플레이 패턴 관찰
   - 움직임 예측
   - 전략 대응

3. **패 관리**
   - 좋은 카드를 일찍 모두 사용하지 않기
   - 중요한 순간에 강력한 카드 보유
   - 리스크와 보상 균형

### 고급 전략

#### 1. 카드 카운팅
- 낸 카드 기록
- 남은 카드 확률 계산
- 정보 기반 의사결정

#### 2. 블러핑
- 특정 카드 보유한 척하기
- 상대 실수 유도
- 심리전 사용

#### 3. 리스크 관리
- 층별 리스크 계산
- 안전하게 플레이할 때 알기
- 리스크 감수할 때 알기

#### 4. 콤보 체인
- 여러 카드 효과 조합
- 파괴적인 콤보 생성
- 점수 획득 최대화

### 가디언별 팁
| 가디언 | 전략 |
|--------|------|
| ♠ 스페이드 | 수비적 플레이 |
| ♥ 하트 | 관계 구축 |
| ♦ 다이아몬드 | 부 축적 |
| ♣ 클럽 | 상대 공격 |

## 카드 시스템

### 카드 희귀도
- **커먼** (회색) - 기본 카드
- **언커먼** (녹색) - 강화 효과
- **레어** (파란색) - 강력한 능력
- **에픽** (볍색) - 게임 변경 카드
- **레전더리** (금색) - 궁극의 힘

### 특수 카드

#### 실드 카드
- 분노 효과로부터 보호
- 게임당 1회 제한
- 도난 불가

#### 텔레포트 카드
- 임의 층으로 점프
- 중간 층 스킵
- 전략적 포지셔닝

#### 스왑 카드
- 상대와 위치 교환
- 역전 플레이에 적합
- 중요한 순간에 사용

#### 더블 카드
- 다음 매치가 2회 분
- 등반 가속
- 다른 효과와 콤보

## 자주 묻는 질문

### Q: 카드는 어떻게 얻나요?
**A:** 다음 방법으로 카드를 획득할 수 있습니다:
- 게임 승리
- 일일 퀘스트 완료
- 카드 팩 개봉
- 다른 플레이어와 교환
- 토너먼트 보상

### Q: 연결 끊기면 어떻게 되나요?
**A:** 
- 30초 이내: 자동 재연결
- 30-60초: AI가 일시적으로 대행
- 60초 후: 기권으로 처리

### Q: 친구와 함께 플레이할 수 있나요?
**A:** 네! "친구 초대" 기능으로:
- 프라이빗 룸 생성
- 커스텀 규칙 설정
- 랭크 관계없이 함께 플레이

### Q: 랭킹 시스템은 어떻게 작동하나요?
**A:** 
- 승리로 랭킹 포인트 획득
- 패배로 포인트 감소
- 높은 랭크 = 좋은 보상
- 시즌별 랭크 리셋

### Q: 게임 내 구매가 있나요?
**A:** 
- 외형 아이템만 (테마, 아바타)
- 페이투윈 요소 없음
- 모든 카드는 게임플레이로 획득 가능

### Q: 토너먼트는 어떻게 작동하나요?
**A:** 
- 참가비가 있는 정기 이벤트
- 토너먼트 형식의 토너먼트
- 상금 풀 분배
- 랭킹 포인트 복수
