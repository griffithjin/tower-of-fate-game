# 🚀 Unity项目快速启动 - 占位符版本

## 说明
这是快速开发的占位符版本，使用纯色方块代替美术资源。

## 颜色规范

### 卡牌颜色
- 红桃 ♥: 红色 (#FF0000)
- 方块 ♦: 橙色 (#FF8000)
- 梅花 ♣: 黑色 (#000000)
- 黑桃 ♠: 黑色 (#000000)

### UI颜色
- 背景: 深灰 (#1A1A2E)
- 按钮: 金色 (#D4AF37)
- 文字: 白色 (#FFFFFF)

## 场景设置

### MainMenu场景
1. 创建Canvas (1920×1080)
2. 添加背景色块 (深灰)
3. 添加按钮:
   - 开始游戏 (金色)
   - 多人对战 (金色)
   - 商店 (金色)
   - 设置 (金色)
   - 成就 (金色)
   - 排行榜 (金色)
   - 好友 (金色)
4. 添加玩家信息区域

### Gameplay场景
1. 创建Canvas
2. 添加背景色块
3. 添加4个玩家面板 (屏幕四角)
4. 添加守卫卡牌区域 (屏幕中央)
5. 添加手牌区域 (屏幕底部)
6. 添加倒计时显示
7. 添加出牌按钮

## 预制体清单

### 必需预制体
1. Card.prefab - 卡牌预制体
   - Image (白色背景)
   - TextMeshPro (花色+点数)
   - Button组件

2. PlayerPanel.prefab - 玩家面板
   - Image (背景)
   - TextMeshPro (玩家名)
   - TextMeshPro (层数)
   - TextMeshPro (手牌数)

3. Button.prefab - 通用按钮
   - Image (金色背景)
   - TextMeshPro (文字)
   - Button组件
