## 🔧 使用流程

### 创建团队对战
1. 进入团队战界面
2. 选择对战模式 (2v2/3v3/4v4/5v5)
3. 创建团队 (如果还没有)
4. 创建房间 → 获得房间码
5. 分享房间码给对手
6. 等待对手加入
7. 开始游戏

### 加入团队对战
1. 进入团队战界面
2. 输入房间码
3. 点击"加入"
4. 等待开始

### 快速匹配
1. 进入团队战界面
2. 选择对战模式
3. 点击"快速匹配"
4. 系统自动寻找对手
5. 匹配成功自动开始
