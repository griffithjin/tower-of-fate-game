### 文件大小限制

- **最大文件大小**: 500KB
- **推荐文件大小**: < 200KB (更快的加载速度)
- **自动压缩**: 启用 (超过限制时自动压缩)

### 格式限制

| 格式 | 支持 | MIME类型 |
|------|------|----------|
| JPEG | ✅ | image/jpeg |
| PNG | ✅ | image/png |
| GIF | ✅ | image/gif |
| WebP | ✅ | image/webp |
| BMP | ❌ | - |
| TIFF | ❌ | - |
| SVG | ❌ | - |

### 自动处理功能

1. **自动压缩**
   - 超过500KB的图片自动压缩
   - 保持原始宽高比
   - 质量降低至85%

2. **智能裁剪**
   - 支持手动选择裁剪区域
   - 保持推荐尺寸比例
   - 预览裁剪效果

3. **格式转换**
   - 自动转换为WebP (浏览器支持时)
   - 保持原始格式作为后备

## 链接配置

### 跳转链接

```
必填项: 是
格式: 完整URL (https://...)
示例: https://example.com/landing-page
```

### 打开方式

| 选项 | 说明 | 使用场景 |
|------|------|----------|
| 新窗口/标签页 (_blank) | 在新标签页打开链接 | 外部链接、推广页面 |
| 当前窗口 (_self) | 在当前页面打开链接 | 内部跳转、游戏内页面 |

### 追踪参数

#### 自动追踪参数

启用追踪后，自动附加以下参数：

```
utm_source=fate_tower
utm_medium={placement_id}
utm_campaign={campaign_id}
utm_content={creative_id}
```

#### 自定义追踪参数

```
格式: key1=value1&key2=value2
示例: campaign=spring_sale&creative=banner_a&audience=new_users
```

#### nofollow 属性

- **启用**: 告诉搜索引擎不要追踪此链接
- **禁用**: 允许搜索引擎追踪 (默认)

## 可视化界面功能

### 拖拽上传

1. 支持多文件拖拽
2. 实时显示上传进度
3. 文件验证 (格式、大小)
4. 缩略图预览

### 实时预览

1. **图片预览**: 显示上传后的图片效果
2. **尺寸检测**: 实时显示图片尺寸与推荐尺寸对比
3. **广告模拟**: 在实际广告容器中预览效果
4. **响应式预览**: 模拟不同设备尺寸

### 尺寸检测提示

```
✅ 符合要求: 图片尺寸与推荐尺寸匹配
⚠️ 尺寸不匹配: 图片尺寸与推荐尺寸差异 > 10%
❌ 格式错误: 不支持的图片格式
❌ 文件过大: 超过500KB限制
```

### 批量上传

- 支持同时上传多张图片
- 每张图片独立验证
- 批量设置链接和标题
- 批量保存

## API 接口

### 上传图片

```http
POST /api/ads/upload
Content-Type: multipart/form-data

Parameters:
- file: 图片文件 (required)
- placement_id: 广告点位ID (required)
- auto_compress: 是否自动压缩 (optional, default: true)

Response:
{
  "success": true,
  "data": {
    "url": "https://cdn.example.com/ads/image.jpg",
    "width": 728,
    "height": 90,
    "size": 45023,
    "format": "webp"
  }
}
```

### 创建广告活动

```http
POST /api/ads/campaigns
Content-Type: application/json

{
  "name": "新手引导推广",
  "type": "banner",
  "placement_id": "main-menu-banner",
  "image_url": "https://cdn.example.com/ads/banner.jpg",
  "link": {
    "url": "https://example.com/tutorial",
    "target": "_blank",
    "tracking": true,
    "nofollow": false,
    "params": "campaign=new_user"
  },
  "schedule": {
    "start": "2025-03-08T00:00:00Z",
    "end": "2025-04-08T00:00:00Z"
  },
  "budget": {
    "daily": 1000,
    "cpm": 5
  }
}
```

## 使用示例

### 创建横幅广告

1. 选择点位: 主菜单底部横幅 (728x90)
2. 上传图片: 拖拽或点击选择 (JPG/PNG, <500KB)
3. 设置链接: https://example.com/offer
4. 打开方式: 新窗口
5. 启用追踪: ✅
6. 预览效果: 确认显示正常
7. 保存活动

1. 选择点位: 胜利结算页
2. 上传视频: 选择MP4文件 (<10MB)
3. 设置奖励: 50 宝石
4. 设置链接: 奖励领取页面
5. 预览效果: 测试视频播放
6. 保存活动
