## 📱 病毒传播机制

### 邀请系统

**邀请奖励（双边奖励）：**
- 邀请者：获得限定皮肤
- 被邀请者：获得新手礼包
- 额外奖励：被邀请者消费分成10%

**邀请方式：**
- 分享海报（带二维码）
- 邀请链接
- 面对面扫码

### 分享机制

**分享场景：**
1. 获得完美匹配 → 分享炫耀
2. 升级段位 → 分享成就
3. 获得稀有皮肤 → 分享展示
4. 获胜通关 → 分享战绩

**分享奖励：**
- 每次分享：+10积分
- 每日上限：5次
- 被点击：额外+5积分

### 社交功能

**好友系统：**
- 添加好友
- 查看好友战绩
- 好友排行榜
- 邀请好友对战

**战队系统：**
- 创建/加入战队
- 战队赛
- 战队排行榜
- 战队专属奖励
