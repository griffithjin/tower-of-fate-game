## 1. 通用规范

### 分辨率标准
- **UI设计：** 1920×1080（16:9基准）
- **移动端适配：** 自适应，最低720p
- **卡牌素材：** 512×768 px
- **背景图：** 1920×1080 px（可滚动）

### 色彩规范
- **主色调：** 金色 (#D4AF37) - 代表命运与财富
- **背景色：** 深灰 (#1A1A2E) - 神秘氛围
- **文字色：** 白色 (#FFFFFF) + 金色描边
- **强调色：** 红色 (#E74C3C) - 警告/错误

### 字体规范
- **中文标题：** 思源宋体 / 站酷高端黑
- **中文正文：** 思源黑体
- **数字/英文：** Roboto / Montserrat
- **特效文字：** 自定义艺术字

## 2. 卡牌设计

### 基础卡牌（52张）

**尺寸：** 512×768 px  
**格式：** PNG（透明背景）

#### 卡牌结构
```
+------------------+
|    [点数]        |
|                  |
|    [花色图案]     |
|                  |
|    [点数]        |
+------------------+
```

#### 花色设计（4套皮肤通用）

**♥ 红桃 (Hearts)**
- 东方：红色桃花
- 奇幻：红色水晶心
- 赛博：红色电路心
- 蒸汽：红色机械心

**♦ 方块 (Diamonds)**
- 东方：金色方胜
- 奇幻：蓝色魔法石
- 赛博：青色数据块
- 蒸汽：黄铜齿轮

**♣ 梅花 (Clubs)**
- 东方：黑色梅花
- 奇幻：绿色荆棘
- 赛博：紫色节点
- 蒸汽：黑色管道

**♠ 黑桃 (Spades)**
- 东方：黑色剑形
- 奇幻：银色长剑
- 赛博：蓝色芯片
- 蒸汽：钢铁钻头

#### 点数设计（A-K）
- A：特殊艺术设计
- 2-10：数字+小花色图案
- J/Q/K：人物插画（4套皮肤不同）

## 4. UI元素规格

### 4.1 按钮设计

**普通按钮：**
- 尺寸：200×60 px
- 圆角：8px
- 状态：正常/悬停/按下/禁用

**卡牌按钮：**
- 尺寸：120×180 px（手牌显示）
- 尺寸：180×270 px（选中状态）

### 4.2 图标设计

**功能图标（64×64 px）：**
- 设置
- 音量
- 退出
- 好友
- 商店
- 背包

**段位图标（128×128 px）：**
- 青铜、白银、黄金、铂金、钻石、星耀、王者、首位好运

### 4.3 特效设计

**出牌特效：**
- 花色匹配：绿色光芒扩散
- 完美匹配：金色粒子爆发
- 失败：灰色消散

**升级特效：**
- 层数提升：塔层发光动画
- 段位提升：全屏庆祝动画

**天命卡特效：**
- 使用动画：卡牌放大旋转
- 效果动画：根据卡类型不同

## 6. 动画规格

### 6.1 卡牌动画

**发牌动画：**
- 时长：0.3秒
- 效果：从牌库飞到手牌区
- 缓动：EaseOut

**出牌动画：**
- 时长：0.2秒
- 效果：手牌飞到中央
- 缓动：Linear

**翻牌动画：**
- 时长：0.5秒
- 效果：3D翻转显示
- 缓动：EaseInOut

### 6.2 UI动画

**按钮悬停：**
- 时长：0.1秒
- 效果：放大1.1倍 + 发光

**弹窗出现：**
- 时长：0.3秒
- 效果：缩放 + 淡入

**倒计时：**
- 最后5秒：红色闪烁
- 最后3秒：心跳动画
