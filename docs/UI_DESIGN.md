# 🎨 UI设计文档 - 首位好运：命运之塔

## 设计原则

### 视觉风格
- **主题：** 神秘、命运、东方仙侠
- **主色调：** 金色(#D4AF37) + 深灰(#1A1A2E)
- **辅助色：** 朱砂红、青花蓝
- **字体：** 思源黑体 + 金色描边

### 界面布局
- **基准分辨率：** 1920×1080
- **适配：** 16:9 为主，支持 4:3 和移动端
- **安全区域：** 四周预留 5%

## 5. 动效规范

### 卡牌动画
| 动画 | 时长 | 缓动 | 描述 |
|------|------|------|------|
| 发牌 | 0.3s | EaseOut | 从牌库飞到手牌区 |
| 选牌 | 0.1s | EaseOut | 放大1.1倍+上移 |
| 出牌 | 0.2s | Linear | 飞到场中央 |
| 翻牌 | 0.5s | EaseInOut | 3D翻转 |
| 匹配成功 | 0.5s | EaseOut | 金色光芒扩散 |
| 完美匹配 | 1.0s | EaseOut | 粒子爆发效果 |

### UI动画
| 动画 | 时长 | 缓动 |
|------|------|------|
| 按钮悬停 | 0.1s | EaseOut |
| 弹窗出现 | 0.3s | EaseOut |
| 倒计时闪烁 | 0.5s | Linear循环 |
