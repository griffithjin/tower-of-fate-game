  - 随机事件 (休息/商店/精英/宝藏/祭坛/赌博)
  - 排行榜系统

#### 3. 每周乱斗 ✅
- **文件:** `server/roguelike_brawl.py`
- **功能:**
  - 12种乱斗规则
  - 每周自动轮换
  - 首通奖励
  - 伤害/排名加成

#### 4. 角色职业系统 ✅
- **文件:** `server/character_class.py`
