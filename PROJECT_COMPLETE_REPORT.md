## 🔧 技术架构

### 服务器端
- **游戏服务器:** Python + WebSocket (端口7778)
- **Web客户端:** HTML5 + JavaScript (端口18080)
- **后台管理:** HTML5 + JavaScript (端口18081)

### 数据存储
- 玩家数据：JSON文件持久化
- 游戏状态：内存中实时管理
- 商城库存：内存中管理

### API接口
- 登录/注册
- 房间管理（创建/加入/离开）
- 游戏控制（开始/出牌/结算）
- 商城（获取商品/购买）
- 锦标赛（获取列表/报名）
- 统计（获取数据/排行榜）
- 每日奖励（获取状态/领取）
