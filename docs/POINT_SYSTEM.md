# Tower of Fate (命运之塔) - 积分体系管理制度

**文档版本:** v1.0  
**创建日期:** 2026年3月3日  
**作者:** 小金蛇

## 1. 货币类型

### 1.1 金币 (Coins)
- **主要用途:** 游戏内基础货币
- **获取方式:**
  - 每日登录奖励: 100-500金币（连续登录加成）
  - 游戏对局奖励: 胜场+100，败场+20
  - 成就解锁奖励: 100-5000金币
  - 锦标赛奖金: 按排名分配
  - 邀请好友: 每邀请1人+100金币

### 1.2 钻石 (Diamonds)
- **主要用途:** 高级货币，购买稀有道具
- **获取方式:**
  - 充值购买 (1元 = 10钻石)
  - 赛季通行证奖励
  - 特殊活动奖励
  - 锦标赛冠军奖励

## 3. 积分获取详细规则

### 3.1 每日登录奖励
```
第1天: 100金币
第2天: 150金币
第3天: 200金币
第4天: 250金币
第5天: 300金币
第6天: 400金币
第7天: 500金币 + 10钻石
```
连续7天后重置，断签则从第1天开始。

### 3.2 游戏对局奖励

#### 基础奖励
- 完成一局游戏: +20金币
- 单局胜利: +80金币（累计100）
- 登顶A层（13层）: +200金币

#### 额外奖励
- 完美匹配: +20金币
- 使用天命牌: +10金币
- 连胜加成:
  - 2连胜: +10%
  - 3连胜: +20%
  - 5连胜: +50%
  - 10连胜: +100%

### 3.3 成就系统奖励

| 成就名称 | 达成条件 | 奖励 |
|---------|---------|------|
| 初露锋芒 | 获得首场胜利 | 500金币 |
| 十胜将军 | 累计10场胜利 | 1000金币 |
| 百战百胜 | 累计100场胜利 | 5000金币 |
| 天命之子 | 累计获得10次完美匹配 | 2000金币 |
| 富豪 | 累计获得10000金币 | 3000金币 |
| 登顶者 | 成功登顶A层 | 500金币 |
| 收藏家 | 拥有10件道具 | 1000金币 |
| 社交达人 | 邀请5位好友 | 2000金币 + 50钻石 |

### 3.4 锦标赛奖金分配

#### 单败淘汰制
- 冠军: 50% 奖金池
- 亚军: 25% 奖金池
- 四强: 12.5% 奖金池
- 八强: 6.25% 奖金池
- 参与奖: 5% 奖金池（平均分配给所有参与者）

## 4. 积分安全与限制

### 4.1 每日上限
- 金币获取上限: 10,000/天
- 钻石获取上限（免费）: 100/天

### 4.2 防作弊机制
- 异常对局检测（AI行为模式分析）
- 刷分账号封禁
- 非法获取金币清零

### 4.3 交易限制
- 金币不可交易
- 钻石不可交易
- 道具不可交易（可购买）

## 6. 经济平衡策略

### 6.1 通货膨胀控制
- 定期调整游戏内奖励
- 控制新金币产生速度
- 增加金币消耗渠道

### 6.2 新用户保护
- 新手任务奖励翻倍（前7天）
- 新手专属商店折扣
- 免费新手礼包

### 6.3 付费点设计
- 性价比递增设计
- 首充双倍奖励
- 月卡/季卡/年卡选项

---

## 7. 运营活动积分规则

### 7.1 节假日活动
- 双倍金币时段
- 限时折扣商店
- 登录送钻石

### 7.2 赛季重置
- 金币不清零
- 钻石不清零
- 赛季专属货币清空

---

## 8. 违规行为处理

| 违规行为 | 处罚措施 |
|---------|---------|
| 使用外挂 | 永久封禁，金币清零 |
| 刷分 | 扣除非法所得，封禁7-30天 |
| 恶意退款 | 扣除相应钻石，限制充值 |
| 账号交易 | 警告/封禁 |
