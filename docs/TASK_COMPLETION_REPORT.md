### 任务1: 修正手牌始终5张 ✅

**问题:** 第一轮出牌后，玩家手中只剩下4张牌

**原因:** `nextRound()` 函数没有调用 `updateDisplay()` 更新手牌显示

**修复:** 在 `web_client/index.html` 第1513行添加:
```javascript
function nextRound() {
    canPlay = true;
    updateDisplay(); // <-- 添加这一行
    startTimer();
    ...
}
```

**测试结果:** ✅ 玩家手中始终显示5张牌
